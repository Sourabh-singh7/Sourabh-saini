Hospital real time bed availability

Overview
Developed for a hackathon at Chandigarh University, this project addresses healthcare accessibility by enabling users to view available hospital beds nearby and access emergency services.

Key Features

  • Bed Availability: Users can check bed availability in nearby hospitals based on their pincode.
  • Online Booking: Patients can reserve beds directly through the web app.
  • Emergency Services: In urgent cases, users can connect with doctors via video or call before the ambulance arrives.
